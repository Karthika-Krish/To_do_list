## My Todo List – Simple Task Management Web Application

## Objective
-The objective of this project is to develop a simple, user-friendly task management web application that helps users organize their daily activities efficiently. The application allows users to add, view, complete, and manage tasks in an organized way.
-This project also helps in understanding core web development concepts such as DOM manipulation, event handling, and local storage using JavaScript.

## Tools and Technologies Used
-HTML5	-> Structure of the application
-CSS	-> Styling and layout design
-JavaScript	-> Application logic and functionality
-Local Storage-> Storing tasks in browser memory
-VS Code	-> Code editor
-GitHub	->Version control and project hosting

## Features
-Add new tasks
-Mark tasks as completed
-View all tasks
-View pending tasks
-View completed tasks
-Clear completed tasks
-Simple and modern user interface

## Working Principle
-The user enters a task in the input field.
-When the user clicks the Add button, the task is added to the task list.
-The task is stored in the browser's Local Storage.
-The user can mark tasks as completed.
-The user can filter tasks as:
-All Tasks
-Pending Tasks
-Completed Tasks
-Completed tasks can also be cleared.
